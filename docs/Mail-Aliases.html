16.5 邮件别名

以下是一个启用邮件别名的 setq，以及更多的提醒。 

;;; 消息模式
; 进入消息模式，输入 'C-x m'
; 进入 RMAIL（阅读邮件），
; 输入 'M-x rmail'
(setq mail-aliases t)

这个 setq 将变量 mail-aliases 的值设置为 t。由于 t 表示真，这一行实际上是说：“是的，使用邮件别名。”

邮件别名是方便的短名称，用于表示长电子邮件地址或电子邮件地址列表。保存别名的文件是 ~/.mailrc。你可以这样写一个别名： 

alias geo george@foobar.wiz.edu

当你给George写一封邮件时，请将地址写为 ‘geo’；邮件客户端将自动将 ‘geo’ 展开为完整的地址。
