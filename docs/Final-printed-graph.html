C.4.6 打印图表

当制作并安装完成后，您可以像这样调用 print-graph 命令： 


(print-graph fiftieth-list-for-graph 50 10)

这是图表：

        **

       *************                 *
  50 - ***************** *           *
       |   |    |    |    |    |    |    |
      10  50  100  150  200  250  300  350


最大的函数组包含每个函数 10–19 个字和符号。
