3.7 特殊形式 if

另一种特殊形式是条件语句 if。该形式用于指导计算机进行决策。您可以编写不使用 if 的函数定义，但它被经常使用且足够重要，因此在这里进行介绍。例如，在函数 beginning-of-buffer 的代码中就使用了它。

if 的基本思想是，如果测试为真，那么就会评估一个表达式。如果测试不为真，则不会评估该表达式。例如，您可能会做出这样的决定：“如果天气暖和而且阳光明媚，那么就去海滩！”
