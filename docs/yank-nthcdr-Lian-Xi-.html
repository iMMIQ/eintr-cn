10.3 使用 yank 和 nthcdr 的练习

  • 使用 C-h v (describe-variable)，查看你的kill ring的值。向kill ring中添加几个项，再次查看其值。使用 M-y (yank-pop)，在kill ring中循环移动。你的kill ring中有多少项？找到 kill-ring-max 的值。你的kill ring是满的吗，还是可以保存更多文本块？
  • 使用 nthcdr 和 car，构建一系列表达式，返回列表的第一、第二、第三和第四个元素。
